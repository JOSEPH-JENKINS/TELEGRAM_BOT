מדריך שימוש: 


- נכנסים לאתר - https://my.telegram.org/auth

- יוצרים Telegram App

- ממלאים את המשתנים בסקיפט שנשלח אליכם (Api Id, Api Hash ומספר טלפון כולל קידומת).

- במשתנה groups רושמים את הקבוצות בהם תרצו לפרסם ללא @ או כתובת מלאה.
   זאת אומרת שאם הקישור של הקבוצה הוא https://t.me/group אתה רושם "group".

- צור ערוץ/קבוצה ותכין בה את כל הפרסומים שתרצה לפרסם, ותרשום את קישור הקבוצה
  (כמו שתיארתי למעלה ללא שטורדל וקישור מלא) במשתנה forward_from.

- שומרים את השינויים ומריצים את הבוט בשורת הפקודה ע"י: python3 bot.py

Instructions for use:


- Visit the site - https://my.telegram.org/auth

Creating a Telegram App

- Fill in the variables in the shift sent to you (Api Id, Api Hash and phone number including prefix).

- In the variable groups, register the groups in which you would like to publish without @ or a full address.
   This means that if the group's link is https://t.me/group you enter "group".

- Create a channel / group and prepare in it all the publications you would like to publish, and write the group link
  (As I described above without shuffle and full link) in the forward_from variable.

- Save the changes and run the bot at the command prompt by: python3 bot.py
